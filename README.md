# 운영체제 (Operating Systems)

운영체제 수업에서 수행한 과제 모음입니다.

## 과제 목록

### 1. [OS CPU Scheduler Simulator](./os-scheduler/)

8가지 CPU 스케줄링 알고리즘(FCFS, SJF, LJF, Priority, Round Robin 등)을 시뮬레이션하고 성능 지표를 비교하는 C 프로그램입니다.

### 2. [SWU Shell](./shell-program/)

`fork`/`execvp` 기반의 간이 Unix 셸 프로그램입니다. 명령어 실행, 입출력 리다이렉션, 파이프를 지원합니다.
